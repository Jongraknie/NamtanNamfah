# ห้องเรียนอนาคตใหม่ (Future Classroom) 🏫
ห้องเรียนสื่อการสอนออนไลน์สำหรับ ป.2 จัดทำขึ้นโดยเฉพาะสำหรับ น้องน้ำฟ้า 💖

## 🎮 รายละเอียดวิชาและเกมนวัตกรรม
1. **คณิตศาสตร์ อนาคตใหม่ (Math Adventure):**
   * 🥕 **ตะลุยป่าแครอท:** ฝึกบวกเลขสามหลักไม่เกิน 1000 (มีทด/ไม่มีทด)
   * 🍦 **ร้านไอศกรีมพ่อมด:** ทายค่าประจำหลักเลขโดด และรูปกระจายจำนวนเลขโดด
   * 🪙 **ล่าสมบัติใต้บาดาล:** เปรียบเทียบค่าของจำนวน และเติมค่าอนุกรมตัวเลข
2. **ภาษาอังกฤษ ป.2 (English Classroom):**
   * 🐡 **เล่ม 1 Coral Reef Adventure:** ทบทวนคำศัพท์ทั่วไป สี และไวยากรณ์พื้นฐาน (have/is/These/an/those/on/in/under/that)
   * 🏫 **เล่ม 2 The School Life:** ฝึกคำศัพท์สิ่งของ บทสนทนาในโรงเรียน ท่ามกลางปลาทะเลแอนิเมชันเคลื่อนไหวสมจริง
3. **เขาวงกตปริศนา (Neon Maze Explorer):**
   * 🌀 **เขาวงกต Cyberpunk:** ฝึกฝนทักษะการวางแผนและการนำทางผ่าน D-Pad เก็บคะแนนสูงสุดเพื่อคว้าตแหน่ง **พระราชา/ราชินี** ป้องกันกำแพงเดินชนมีเอฟเฟกต์เสียงสังเคราะห์
